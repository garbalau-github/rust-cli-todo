### Terminal Todo

This is an interactive terminal todo application in *Rust* with help of ncurses, a very thin wrapper around the ncurses TUI library, which gives control over terminal interface

### Quick Start

```console
cargo run TODO
```

### Controls

| Keys             | Description                     |
| ---------------- | ------------------------------- |
| <kbd>q</kbd>     | Quit                            |
| <kbd>w</kbd>     | Move Up                         |
| <kbd>s</kbd>     | Move Down                       |
| <kbd>Tab</kbd>   | Switch tabs (TODO and DONE)     |
| <kbd>Enter</kbd> | Action (on highlighted element) |
