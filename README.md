### Terminal Todo

Interactive Terminal Todo Application in Rust

### Quick Start

```console
cargo run
```

### Controls

| Keys             | Description                     |
| ---------------- | ------------------------------- |
| <kbd>q</kbd>     | Quit                            |
| <kbd>w</kbd>     | Move Up                         |
| <kbd>s</kbd>     | Move Down                       |
| <kbd>Tab</kbd>   | Switch tabs (TODO and DONE)     |
| <kbd>Enter</kbd> | Action (on highlighted element) |
